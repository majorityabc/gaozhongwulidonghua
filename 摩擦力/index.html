摩擦力演示：小孩推箱子（μ = 0.30，g = 10 N/kg）

空箱 f_max = 60 N 能推动；加一块石块 f_max = 90 N 刚好还能推动；两块石块 f_max = 120 N 就推不动了
推力 F： 0N 小孩最多能使出 100 N 的力气。F ≤ f_max 时箱子纹丝不动；F 一旦超过 f_max，箱子开始匀加速滑动
当前阶段：
最大静摩擦力：
推力 F = N
当前摩擦力 f静 = N
状态：
说明：静摩擦力是个“被动力”——推力多大它就多大（方向相反），所以推力没超过最大静摩擦力时箱子纹丝不动，二力平衡。最大静摩擦力 f_max = μN = μ·m总·g，箱子越重越难推动。一旦推力超过 f_max，箱子开始滑动，此后滑动摩擦力保持 μ·m总·g 不再随推力增大，多出来的那部分推力成为合力，让箱子加速。往箱子上加东西，压力 N 变大，f_max 跟着变大，就需要更大的推力才能推动了。
